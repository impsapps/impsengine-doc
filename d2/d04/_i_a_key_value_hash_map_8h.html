bool IAKeyValueHashMap_load ( const IAHashMap *  elements,
const IAString *  data 
)

Load all values from data into the corresponding key value delegates.

If the hash map does not contain a delegate for a key, the key value set will be skipped.

Parameters
elementsMust only include key value delegates.
dataThe data to get load into the key value delegates. This must be created via "IAKeyValueHashMap_save()". However, the data object can be extended. The additional extension is ignored. This is useful if you want to use a cryptographic encryption algorithm where padding is needed.
Returns
Returns true if the data is valid and well formatted, false otherwise.
